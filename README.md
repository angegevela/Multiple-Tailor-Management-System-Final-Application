# Multiple Tailor Shops Management System in Puerto Princesa City
Capstone Research (Thesis) Project

## Overview

This project is a starting point for this Flutter application.
Thus, this repository consist of recents codes, databases(Firebase & Supabase Connections), and the extensions used on this project.

## Objectives/Motivation Making This Project

This project aims to create an integrated system solution for tailoring shops in Puerto Princesa City, addressing their scheduling and workload management challenges. The system enhances service efficiency and customer satisfaction by enabling users to identify nearby tailors and book appointments online, reducing waiting times. Key features include managing appointment scheduling, customer data, and workload organization, thereby improving business operations and ensuring maximum efficiency across shops. This initiative is expected to transform the operational management of tailor shops, fostering both increased efficiency and improved customer experiences.

## Methodology

This project utilizes Flutter and Android throughout the Thesis Timeline.

## Key Findings

The Multiple Tailor Shops Management System was developed to tackle challenges in the tailoring industry in Puerto Princesa City, such as manual appointment handling, workload imbalance, and communication issues between tailors and customers. The researchers gathered qualitative and quantitative data through user testing and interviews with 9 tailor shops and 11 customers. These responses informed a qualitative analysis that highlighted the key findings from the interviews.
